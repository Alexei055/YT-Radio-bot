Привет, мой дорогой друг! Хочу тебя поблагодарить за скачивание моего бота и рассказать как его настроить.
Следуй следующим пунктам настройки бота и пользуйся им!
----

0. Установи Python 3.6.0 и несколько модулей для работы бота [ПРОПУСТИТЕ ВСЕ ПУНКТЫ ОТНОСИТЕЛЬНО #0 ЕСЛИ У ВАС УЖЕ УСТАНОВЛЕН PYTHON C DISCORD.PY!!!]
0.1.1 Скачай установочный файл по этой ссылке: https://www.python.org/ftp/python/3.6.0/python-3.6.0-amd64.exe
0.1.2 Открой файл python-3.6.0-amd64.exe и установи Python предварительно добавив его в PATH (https://imgur.com/eyvhmgU)
0.1.3 Закончи установку и следуй остальным пунктам
0.2.1 Открой команную строку (Ctrl+R, cmd или найти cmd в поиске)
0.2.2 Введи pip install discord.py --user и ожидай окончания установки
0.2.3 Введи pip install youtube-dl --user и также ожидай окончания установки 
0.2.4 Введи pip install PyNaCl --user и опять же ожидай окончания установки
----

1. Создай своё приложение на сайте https://discordapp.com/developers/applications/  [ПРОПУСТИТЕ ВСЕ ПУНКТЫ ОТНОСИТЕЛЬНО #1 ЕСЛИ У ВАС УЖЕ СОЗДАНН АККАУНТ БОТА!!!]
1.1 Создай аккаунт бота во вкладке "Bot" нажатием на кнопку "Add bot" (https://imgur.com/g9NeJHt)
1.2 Задай имя и иконку (необязательно) и скопируй токен бота чтобы использовать его в пункте #3 (https://imgur.com/1ZN5hMl)
1.3 Пригласи бота на свой сервер по ссылке, которую вы сгенерируете на сайте во вкладке "OAuth2"
1.3.1 Поставь галочку Bot и нажми на кнопку "Copy" чтобы скопировать ссылку, затем перейди по ней и добавь бота на сервер, который будет указан в списке (https://imgur.com/Yrf57ou)
----

2. Открой файл config.py
3. Заполни все поля, которые не имеют пометки "необязательно" (необязательные поля можно заполнить по желанию)
4. Сохрани изменённый файл
5. Запусти файл index.py

[ДЕРЖИТЕ ВСЕ ФАЙЛЫ БОТА В ОДНОМ МЕСТЕ, ИНАЧЕ БОТ НЕ БУДЕТ РАБОТАТЬ!!!]
